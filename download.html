Configuration Prérequis pour l'Exécution

Afin de faire tourner le script Ursina et d'éviter les crashs de rendu lors de la génération de la carte, vérifiez vos modules :

  • Système d'exploitation : Windows 10 ou 11 (64 bits).
  • Environnement : Python 3.9 ou supérieur requis si lancé depuis les fichiers sources.
  • Librairies Python requises : ursina, perlin-noise, PyQt5, deep-translator, sqlalchemy.
  • Contrôles : Clavier ZQSD (paramétrable) ou Souris dotée d'une molette fonctionnelle pour le défilement de l'inventaire et des menus déroulants.
