Configuration Prérequis pour l'Exécution (Installées automatiquement dans l'éxécutable)

Afin de faire tourner le script Ursina et d'éviter les crashs de rendu lors de la génération de la carte, vérifiez vos modules :

  • Système d'exploitation : Windows 10/11(64 bits).
  • Environnement : Python 3.9 ou supérieur requis si lancé depuis les fichiers sources.
  • Librairies Python requises : ursina, perlin-noise, deep-translator, sqlalchemy.
  • Contrôles : Clavier ZQSD (paramétrable) et Souris dotée d'une molette fonctionnelle pour le défilement de l'inventaire et des menus déroulants.
